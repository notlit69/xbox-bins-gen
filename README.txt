Node.js v18.14.1
brain cells
xbox bins
msft accs

Usage : 
Put proxies in config.json (either host:port or email:pwd@host:port) or leave empty for proxyless
Find the locale of your desired game pass location by finding the gamepass URL. 
For example, if you want to use a UAE bin search xbox game pass arab emirates and find the URL.
Your URL should be something like this : https://www.xbox.com/ar-AE/xbox-game-pass. 
Here your locale is ar-AE.
Now, Find any random zip code/ postal code for your BIN country (A valid one) and use that in config
At last, put your BIN in the config eg. for a bin "411234567854xxxx|02|2027|000"
Put emails in accs.txt email|pass format.
run the start.bat

After you have got some gamepass accs. If you want to fetch the 3m promo links heres a guide:
open the promo-server file open a cmd prompt in the folder and run:
npm i
node server.js
Make sure the server is running!
Now, open the promo-puller file put your gamepass accs in accs.txt and run the main.py to fetch the promos.
